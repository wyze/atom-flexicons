> A flexible full-color icon package for use in Atom.

## Install

```shell
$ apm install flexicons
```

## Develop

```shell
$ git clone https://github.com/wyze/atom-flexicons.git
$ cd atom-flexicons
$ npm install
$ apm link -d
$ atom . -d
```
